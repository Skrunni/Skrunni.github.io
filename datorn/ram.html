RAM. Random Acces Memory. För att datan ska kunna komma ihåg saker och för att du ska kunna använda många saker på datan samtidigt är RAM viktigt att ha. Om du vill liksom använda hundra tusen webbsidor samtidigt behöver du ett bra RAM som kan göra det möjligt. Med bättre RAM blir datan ganska snabbare också, när du klickar på någonstans laddar det upp snabbare och datan svarar snabbare till dina kommando. Utan RAM kommer datan inte ens kunna göra något typ, allt som du säger till den att göra kommer hellre ta förevighet att hända eller den kommer inte ens kunna göra det.
